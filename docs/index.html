О проекте

EPD - это инструмент, который позволяет пользователю легко загружать программы и скрипты с GitHub и других ресурсов.

С помощью EPD вы можете автоматизировать процесс скачивания программ и получать доступ к необходимым инструментам в одно мгновение.
